# DigitalSanctuary — iOS Mood Tracker

> ⚠️ Work in progress / vibe-coded project. Not production ready.

A SwiftUI mood tracker app for iOS. Track your daily mood with emojis, short reflections, and photos. View trends monthly and weekly.

## Features
- **Daily view** — pick a mood emoji, write a reflection (200 chars), add up to 4 photos
- **Weekly view** — see your week at a glance with a fun AI-free summary
- **Monthly view** — calendar grid, mood cloud, trend stats, and encouraging quotes on tough months

## Mood Types
| Emoji | Mood |
|-------|------|
| 😢 | Sad |
| 😶 | Neutral |
| 😊 | Happy |
| 🌱 | Grounded |
| 😠 | Frustrated |
| 😨 | Anxious |
| 💅 | Fabulous |

## Tech Stack
- SwiftUI + SwiftData
- iOS 17+
- No third-party dependencies
